Crankshaft

クランクシャフト

クランク・ピン　→ コンロッド

クランクジャーナル　→ シリンダブロック

「ねじれ」に対す抗力　→ 短くて太い

軽い　→ 中空など

耐摩耗性　→ 表面硬化処理（高周波焼き入れ）

材料　→ 炭素鋼、特殊鋼、特殊鋳鉄

バランス・ウェイト　→ クランクシャフト回転部分の質量バランスを修正する

クランクシャフトは、ピストンの往復運動をコンロッドを介して回転運動に変えるため、すなわち、ピストン位発生したエネルギを動力として取り出すもので、クランクピン、クランクジャーナル、クランクアーム、バランスウェイトなどで構成されている。

バランスウェイトに「穴」がある　→ バランスを取るため

クランクシャフトは、大きな荷重を受けながら高速で回転するため、強度、剛性及び耐摩耗性が大きく、さらに、静的、動的バランスがとれ、円滑に回転することが必要である。

クランクシャフトの材料はには、一般には炭素鋼、あるいは、特殊鋼などが用いられている。加工方法は、型打鍛造した青戸に機会に加工され、クランクジャーナル及びピンには、耐摩耗性を高めるため、表面硬化処理が施されている。

バランスウェイトには、ランダムにドリル穴が開けてあるのが散見されると思うが、これはクランクシャフちの動的バランスを取るための重量調整した部分である。穴の深さも位置もバラバラである。

フィレット・ロール加工　→ 応力集中を避ける　→ ジャーナルとピンはオーバーラップさせる必要がある。

クランクシャフトには、軸心に対するアンバランスをなくすため、バランス・ウェイトにより調整してあり、図のようにクランクジャーナル及びクランクピンお端部は、力の集中を避けるために丸みをつけている。実際のクランクシャフトの写真を次ページ載せてある。丸みをつけているというより、半円程度に凹ましてある。これを、フィレット・ロール加工という。

クランク・ジャーナルは、シリンダブロックとともにクランクシャフトを支えるためのもので、その数は、一般に、直列4気筒の場合は５個、直６なら７個、V８は５個、V１０は６個となっている。（V型エンジンは、向かい合う位置のシリンダ同士は、同一のピンを共用するため）。ピントピンの間、及び両端にジャーナルを持つことになる。

クランクピン及びコンロッドベアリングへの給油を行なっている。

クランクシャ油路

クランク・ジャーナルとクラン・ピンを繋ぐ油路を設けて、クランク・ピン及びコンロッド・ベアリングへの給油を行っている。実際は、この図のような加工しない（できない、ドリルは直角にしないと折れる、遠心力を考慮した開け方にしてある）。内部にオイル通路があるイメージとして見ておいてください。

スタスト・プレート：クランクシャフトの軸方向の力を受け止める。

じゃーなる・ベアリング

コンロッドベアリングと同様な構造で使い方も基本的は同じである。特に、内面にオイルが圧送されてきて十分供給されやすいように、オイル穴とオイル溝が設けられている。また、ジャーナルの一箇所には（4気筒エンジンなら第３ジャーナル）、クランクシャフトの軸方向の力を受けるためのスラスト・ペアリング（スラスト・プレート）が設けられている。スラストとジャーナルを一体化したベアリングもある。

クランクシャフトに働く力

図のようにピストンを下に押す力は（F）は、サイド・スラスト（Fn）とクランクピンに掛かる力（Fc）があったが、Fc＝P１とする。P１はの力は、クランクピンの回転方向の力クランクジャーナル方向の力に分けられ、P２、P3の力が主にクランクシャフトに働く力であり、P２にクランクピンの回転半径（r）を乗じたものが軸トルクである。なお、このほかに、クランクシャフトに働く力として、往復運動部分の慣性力、回転運動部分の遠心力などがあるが、これらは有効な力とはならないで、騒音、振動のもとになっている。また、クランクシャフトには、燃焼圧力によるトルクの変動によってねじり振動が生じる。一般に、長いクランクシャフトでは、エンジンが高速になると、このねじり振動による疲労が起こる。これを防止するためには、クランクシャフトのねじり剛性を大きくすることは、エンジンの軽量化、小型化などの面から制限がある。したがって、一般にクランクシャフトの長いエンジンでは、トーショナル・ダンパを装着している。

ねじり振動を吸収するため　→

ラバー・ダンパ式　→ 構造簡単、安価、振動抑制効果　小

ビスカス・ダンパ式　→ シリコンオイル充填　→ 振動抑制効果　高

ピスカス・ラバー・タンパ式　→ シリコンオイル充填　→ 振動抑制効果　中

トーショナル・ダンパ（クランクプーリ）

プリと一体になったトーショナルダンパは、図のようになっており、クランクシャフトが一定回転速を保って回転している時であれば、クランクシャフトとプーリは一体回転している。しかしクランクシャフトにねじり振動が発生した時（回転数が増やした時又は減ったとき）は、プーリはそのまま一定速度で回転し続けようとするため、トーショナル・ダンパにより、ねじり振動の減衰作用が割れる。

他には、ラバー・ダンパ式（プーリタイプと同形式）、ビスカス・ダンパ式、ビスカス・ラバー・ダンパ式がある。

フライホイール

クランクシャフトにかかる回転力を平均化する

クラッチへ動力を伝達する

慣性力：止まっているものは止まり続け、働いているものは働き続けようとする力。

リングギヤ：スタータのピニオンと噛み合い、スタータの力をエンジンに伝える。

マニュアル・トランスミッション車などで使用されるフライホイールは、図に示すようなもので、４サイクルエンジンでは２回転毎の燃焼によって変化する回転力を平均化する働きと、クランクからクラッチへ動力を伝達する役目を果たしている。すなわち、クランクシャフトの回転力が燃焼行程によって増加したときは、そのエネルギをフライホイールが吸収し、燃焼以外の行程でクランクシャフトの回転力が減少した時には、フライホイールの完成エネルギによって回転を維持して、クランクシャフトの回転速度の変化を少なくし、円滑な回転状態を維持させている。フライホイールは鋳鉄製で、大きさは、エンジン出力、回転速度及びシリンダ数によって異なるが、かなり大きな長径と質量を持ち、クランクシャフト後端部に取り付けられ、クラッチ・ディスクとの摩擦面は柔らかに仕上げられている。また、フライホイールの中心には、パイロットベアリングがはめ込まれ、クラッチシャフトの一端を支えるようになっている。リング・ギアは、エンジンを始動するとき、スタータのピニオンと噛み合い、スタータの回転をフライホイールに伝えるためのもので、一般に炭素鋼製のスパー・ギアが用いられ、フライホイールの外周に焼き嵌めされている。リング・ギアの歯先は、焼き入れを施して耐久性の向上を図るとともに、スタータのピニオンのかみ合いを容易にするため片側は面取りしてある。なお、フライホイール外周には、シリンダ番号と角度目盛りが刻印されているものもある。

トルクコンバータ装着車（AT車）　→ ドライブプレートを使用　→ トルコンがフライホイールの役目をする（重量がある）

オートマチック・トランスミッション車などで使用されるトルク・コンバータは、トルク・コンバータがフライホイールの役をするため、前述したフライホイールは設けられていない。そのため、軽量で薄い図のようなドライブプレートをクランクシャフトの後端に取り付けている。リング・ギアは、ドライブ・プレートに点付き溶接で取り付けてある。

クランクシャフトに発生するいらない力　→ バランス・ウェイト、フライホイール、トーショナル・ダンパ　→ 吸収しれない　→ 騒音、振動になる　→ バランサ機構で吸収する

一次慣性力（エンジン１回転　＝　１サイクルの慣性力）　→　クランクシャフトのバランスウェイトで低減

二次慣性力（エンジン一回転　＝　２サイクルの慣性力）　→ バランスシャフトで低減

前述したように、クランクシャフトには色々な力が加わり、それらは振動や騒音の原因でおある。これまで、バランス・ウェイト、トーショナル・ダンパ、フライホイールなど解説したが、それらだけでは取り去ることのできない振動源がある。それを吸収する仕組みにに、バランサ機構がある。そもそも、エンジンが回転することにより発生するものを一次慣性力と言い、エンジンから出てくる一次慣性力は複数上がるので、それらが合成されたものを二次慣性力という。二次慣性力は、バランス・シャフトで低減させることができる。

クランクシャフトの２倍の速度で回転

図は、4気筒エンジンのバランサ機構の一例である。クランクシャフトと並列に配置された２本のバランス・シャフトおよびこれを駆動するタイミングギアなどで構成されており、バランスシャフトは、クランクシャフトの２倍の回転速度で、それぞれ図に示す矢印の方向に回転するようになっている。２本のバランスシャフトはお互いに逆回転するようになっている。

２本必要な理由、逆回転する理由、２倍の回転速度で回す理由を次に示す。

エンジンの振動源は、ピストンやコンロッドが往復するときに発生する往復運動部分の慣性力である。この慣性力の低減方向として、クランクシャフトにバランス・ウェイトを設けており、これにより一次慣性力（エンジン一回転に１サイクルの慣性力が発生すること）を低減させている。しかし、図のようにピストンの最高速度到到着点は、ストロークの中間より上してん側に近い位置にあるため、この時発生する慣性力は、図のように、ピストン正方向の慣性力と負方向の慣性力は全く同じにならず、正方向の慣性力の方が負方向より大きくなる。

なので、ピストンが往復運動をする時の上下方向の力はアンバランスな状態となる。最高速到着点において、クランク中心、クランク・ピン、ピストン・ピンを結んだ三角形は直角三角形となる。つまり、この瞬間がコンロッドで押してくる力の全てがクランプを回す力として使われる。また、上してんと下してんの中間（ストロークの中間）にピストンが位置するときの三角形、コンロッド長を二辺とする二等辺三角形となる。

一般に４気筒エンジンでは、１番、４番のピストンが同位相にあり（ここでは物理的な位置が問題になり、行程は関係ない）、２番、３番のピストンは１８０度ずれた同位相にある。したがって、これらの慣性力はそれぞれは同じであるが、慣性力の合力は図のAに示す合力は、１、４番の正方向の慣性力と２、３番負方向の慣性力の差、すなわち、FーF｀＝Aとなり、図のBではf＋f｀＝Bとなる、この慣性力の合力は、二次慣性力と呼ばれ、赤実線のようにエンジン一回転につき２サイクルの慣性力となり、バランス・ウェイトでは低減できない。したがった、エンジンの２倍の回転速度で回転するバランスシャフトを設け次ページのように、逆位相の慣性力を発生させて二次慣性力を打ち消します。

先の説明で、バランスシャフトを２倍の回転数で回さなければならない。理由は、わかりましたか。その説明は、ここの図でも分かりますね。エンジンの二次慣性力を、バランスシャフトの一次慣性力で、打ち消すような逆位相している

さて、もう一つ疑問も疑ってますね。なぜ２本必要なのか？ここまでの内容であれば、１本でも成立するんですが２本わざわざ使ってあるのは、こういうことです。１本だと上下方向はピストンnと相殺させることができるが、左右に振動が出てしまいます。そこで、２本にしてバランスシャフトのウェイトが水平方向を向いたときは、必ずが逆方向を向くようにしてあります。
